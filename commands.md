Чтобы проинициализировать Git репозиторий введите команду. Будет создана папка /.git/. Репозиторий - это скрытая папка, в которой работает Git. 
```
git init
```
Чтобы добавить в Git все файлы:
```
git add .
```
Чтобы добавить один конкретный файл:
```
git add text.txt
```
Посмотреть статус
```
git status
```
